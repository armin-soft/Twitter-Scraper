## 🚀 **Quick Setup**

### ⚡ **One-Command Installation**

```bash
git clone https://github.com/armin-soft/Twitter-Scraper.git
cd Twitter-Scraper
./Auto-Run.sh
```

The installer automatically:
- **🐍 Installs Python 3.11+** (if needed)
- **📦 Installs dependencies** from requirements.txt
- **🌐 Sets up browser** (Chrome/Chromium)

## 📖 **Usage**

### 🎯 **Basic Usage**

1. **Launch**: `./Auto-Run.sh` or `python Run.py`
2. **Authenticate**: Enter 𝕏/Twitter credentials securely
3. **Select Target**: Choose username, hashtag, or search query
4. **Configure**: Adjust extraction parameters
5. **Extract**: Monitor real-time progress
6. **Export**: Find results in `Exports/` directory

### 📊 **Extracted Data Fields**

| Field | Type | Description |
|-------|------|-------------|
| **URL** | String | Direct tweet link |
| **ID** | String | Unique tweet identifier |
| **Text** | String | Full tweet content |
| **Retweet Count** | Integer | Number of retweets |
| **Reply Count** | Integer | Number of replies |
| **Like Count** | Integer | Number of likes |
| **Quote Count** | Integer | Number of quotes |
| **Created At** | DateTime | Tweet timestamp |
| **Bookmark Count** | Integer | Number of bookmarks |
| **Is Retweet** | Boolean | Retweet status |
| **Is Quote** | Boolean | Quote tweet status |

## ⚙️ **Configuration**

The `Config.json` file contains all customizable settings:

<details>
<summary><strong>🎯 Scraping Settings</strong></summary>

```json
{
  "scraping": {
    "max_threads": 4,              // Concurrent processing threads
    "headless": true,              // Browser visibility
    "min_delay": 0.8,             // Minimum delay between requests
    "max_delay": 2.0,             // Maximum delay between requests
    "tweet_limit": 200000,        // Maximum tweets to extract
    "retry_attempts": 3,          // Failed request retries
    "timeout": 30                 // Request timeout seconds
  }
}
```
</details>

<details>
<summary><strong>📤 Export Settings</strong></summary>

```json
{
  "export": {
    "output_dir": "Exports",      // Output directory
    "default_format": "csv",      // Export format
    "include_summary": true,      // Generate summary reports
    "max_file_size_mb": 100      // Maximum file size
  }
}
```
</details>

<details>
<summary><strong>⏱️ Rate Limiting</strong></summary>

```json
{
  "rate_limiting": {
    "enabled": true,              // Enable rate limiting
    "base_delay": 1.0,           // Base delay seconds
    "requests_per_minute": 45,   // Request rate limit
    "adaptive_delay": true,      // Dynamic adjustment
    "max_delay": 15.0           // Maximum delay seconds
  }
}
```
</details>

<details>
<summary><strong>🔍 Filtering Options</strong></summary>

```json
{
  "filter": {
    "filter_retweets": false,     // Exclude retweets
    "min_likes": 0,              // Minimum like count
    "min_retweets": 0,           // Minimum retweet count
    "filter_duplicates": true,   // Remove duplicate tweets
    "min_text_length": 1,        // Minimum text length
    "max_text_length": 2000      // Maximum text length
  }
}
```
</details>

### 🐛 **Common Issues**

<details>
<summary><strong>Installation Problems</strong></summary>

**Issue**: Auto-Run.sh fails
```bash
# Manual installation
python -m pip install --upgrade pip
pip install -r requirements.txt
python Run.py
```

**Issue**: Browser not found
```bash
# Ubuntu/Debian
sudo apt install google-chrome-stable
# macOS  
brew install --cask google-chrome
```
</details>

<details>
<summary><strong>Authentication Issues</strong></summary>

- Verify 𝕏/Twitter credentials are correct
- Check if account is not suspended
- 2FA is currently not supported
- Try using application passwords if available
</details>

<details>
<summary><strong>Rate Limiting</strong></summary>

- Increase delays in Config.json
- Reduce `requests_per_minute` value
- Enable `adaptive_delay` feature
- Use proxy rotation if needed
</details>
